# Markdown Guide

Learn how to write beautiful content using Markdown syntax.

## Basic Syntax

### Headers

# H1
## H2
### H3

### Emphasis

*italic* or _italic_
**bold** or __bold__
***bold italic*** or ___bold italic___

### Lists

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

1. First item
2. Second item
3. Third item

## Advanced Features

### Code Blocks

```typescript
interface User {
  name: string;
  age: number;
}
```

### Tables

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
